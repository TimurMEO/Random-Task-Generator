# Random Task Generator

**Автор:** [Али Байсултанов]
**Язык:** Python 3.x

## Описание

Простое графическое приложение (GUI) для генерации случайных задач из предопределённого или пользовательского списка. Приложение позволяет сохранять историю сгенерированных задач и добавлять новые.

## Возможности

* Генерация случайной задачи по нажатию кнопки.
* Встроенный список задач по категориям (учёба, работа, спорт, отдых).
* Добавление пользовательских задач с указанием типа.
* История задач: все сгенерированные задачи сохраняются в файл.
* Фильтрация истории по типу
