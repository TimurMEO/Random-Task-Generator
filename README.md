# Random Task Generator

**Автор:** [Балаев Тимур]  
**Язык:** Python 3.x  
**GUI‑библиотека:** Tkinter  
**Хранение данных:** JSON  

## Описание

Простое графическое приложение (GUI) для генерации случайных задач из предопределённого или пользовательского списка. Приложение позволяет сохранять историю сгенерированных задач и добавлять новые, а также фильтровать их по типам.

## Возможности

* Генерация случайной задачи по нажатию кнопки.
* Отображение истории всех сгенерированных задач.
* Фильтрация задач по типам: «учёба», «спорт», «работа», «отдых».
* Добавление новых задач пользователем с указанием типа.
* Сохранение истории в файл `tasks.json`.
* Автоматическая загрузка истории при запуске приложения.
* Валидация ввода: проверка на пустую строку при добавлении новой задачи.

## Требования

* Python 3.6 или выше.
* Стандартные библиотеки: `tkinter`, `random`, `json`, `os` (устанавливать не нужно).
